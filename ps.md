Forecasting and/or Nowcasting of Solar Flares using combined Soft and Hard X-ray data from Aditya-L1
Description
Solar flares are sudden, intense bursts of radiation originating from the release of magnetic energy in the solar atmosphere. These events can trigger severe space weather disruptions, impacting Earth-bound satellite communications, GPS navigation, and power grids. Anticipating these disruptions is critical for infrastructure safeguarding.

ISRO's Aditya-L1 mission continuously monitors the Sun from Lagrange Point L1 using two X-ray payloads: SoLEXS (Solar Low Energy X-ray Spectrometer, capturing soft X-rays) and HEL1OS (High Energy L1 Orbiting X-ray Spectrometer, capturing hard X-rays).

This challenge requires participants to build an automated algorithmic pipeline that uses the combined time-series data from SoLEXS and HEL1OS to detect (nowcast) or predict (forecast) these solar flares.

Objectives
To design an automated solar flare detection algorithm for the nowcasting (real-time detection and classification) of solar flares using soft and hard X-ray data.
To build a predictive algorithm for the forecasting of solar flares by identifying precursor patterns in soft and hard X-ray light curves before actual solar flare occurs.
Expected Outcomes
An automated database of nowcasted solar flares, generated by combining soft and hard X-ray light curves.
A trained model capable of forecasting flares with a quantifiable lead time.
Interface that visualizes the X-ray light curves and triggers with visual alerts when a flare is nowcasted or forecasted.
Dataset Required
Primary Data: Aditya-L1 SoLEXS (Level-1) and HEL1OS (Level-1) data, accessible via the ISRO ISSDC PRADAN portal.
Participants can use any other supplementary datasets.
Suggested Tools/Technologies
SoLEXS and HEL1OS data processing software accessible via the ISRO ISSDC PRADAN portal.
Any other open-source tools.
Expected Solution / Steps to be followed to achieve the objectives
Download SoLEXS and HEL1OS from the ISSDC portal. Write scripts to read SoLEXS and HEL1OS time-series light curve data.
Visually understand typical solar flare light curve observed in soft and hard X-rays.
Algorithmic Nowcasting: Build algorithms to detect flares independently in both soft and hard X-rays. Combine these independent catalogues to generate a master catalogue.
Predictive Modelling: Train a time-series model using combined dataset to predict the probability of a flare occurring in the next N minutes.
Evaluation Criteria
Detection of low- and high-class flares
High True Positive Rate and low False Alarm Rate for predictions.
Lead Time of predictions: How many minutes before the flare peak the model triggers an alert
